# Word Calendar Generator

Generate compact Word (.docx) calendar files.

![Screenshot](screenshot.png)

## Use Online

Visit **[unto.me/word-calendar-gen](https://unto.me/word-calendar-gen/)** to generate calendars in your browser.

## Contributing

```bash
npm install
npm run build
node dist/cli.js -o test-output/test.docx
```
